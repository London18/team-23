How to take "In Memory" calls

Here is a brief guide on how to deal with in memory queries over the phone in case Ayesha, Kelly or Debbie aren’t here to take the call which there has been a few instances recently. Please refer to this when taking any in memory calls, especially funeral donation enquiries.
This is to ensure we capture all the information we need so our BOBs receive the best support needed at a difficult time.   The in memory pages on the website have now been updated as well and people can now complete an online form to request a Giving in Memory pack. Please let me know if you have queries with anything but this should make the process a lot simpler for our BOBs! Also, if you feel you would like any one to one phone training on how to take in memory calls, please do come and speak to Ayesha or Kelly!  We’re always happy to help!
